# 🌐 Global Cyber Threat Analytics Dashboard (2015–2024)

## 📌 Short Description

This project presents an interactive **Power BI dashboard** that analyzes global cybersecurity threats from 2015 to 2024. The dashboard visualizes attack patterns, affected industries, financial impact, and response times to help understand evolving cyber threat trends and support data-driven cybersecurity decisions.

---

# 🛠 Tech Stack

The dashboard was developed using the following technologies:

* **Power BI Desktop** – Data visualization and dashboard development
* **Power Query** – Data cleaning and transformation
* **DAX (Data Analysis Expressions)** – KPI calculations and measures
* **Microsoft Excel** – Dataset storage and preprocessing
* **Canva** – Presentation design and visualization storytelling

---

# 📊 Data Source

**Dataset:** Global Cybersecurity Threats (2015–2024)

Source: Kaggle
[https://www.kaggle.com/datasets/atharvasoundankar/global-cybersecurity-threats-2015-2024](https://www.kaggle.com/datasets/atharvasoundankar/global-cybersecurity-threats-2015-2024)

The dataset contains over **3,000 cybersecurity incident records** across multiple countries and industries.

### Key Attributes in the Dataset

* Attack Type
* Country
* Target Industry
* Financial Loss (Million USD)
* Number of Affected Users
* Attack Source
* Security Vulnerability Type
* Defense Mechanism Used
* Incident Resolution Time (Hours)

This dataset enables multi-dimensional analysis of cyber threats, financial impact, and organizational response effectiveness.

---

# ⭐ Features / Highlights

## Business Problem

Cyberattacks are increasing in frequency, complexity, and financial impact across industries worldwide. Organizations often struggle to identify patterns in cyber incidents and understand which sectors, attack types, or vulnerabilities pose the greatest risk.

Without structured visualization, it becomes difficult for researchers, analysts, and decision-makers to interpret cybersecurity data effectively.

---

## Goal of the Dashboard

The primary objective of this dashboard is to:

* Analyze global cyberattack trends from **2015–2024**
* Identify the most common **attack types and sources**
* Understand **which industries are most targeted**
* Evaluate the **financial impact of cyber incidents**
* Measure **incident response efficiency**

The dashboard transforms raw cyber threat data into **interactive and actionable insights**.

---

## Walkthrough of Key Visuals

### 🔢 KPI Cards

Key performance indicators summarize major cybersecurity metrics:

* Total Cyber Attacks
* Total Financial Loss
* Total Affected Users
* Average Resolution Time

These KPIs provide a quick overview of the global cyber threat landscape.

---

### 🌍 Attack Distribution by Country

A bar chart highlights countries experiencing the highest number of cyber incidents, helping identify regions with greater cyber risk exposure.

---

### 🛡 Attack Type Analysis

A donut chart visualizes the distribution of attack types such as:

* Ransomware
* Phishing
* Man-in-the-Middle
* SQL Injection
* Malware

This helps identify the most prevalent cyberattack methods.

---

### 📈 Financial Loss Trends

A line chart illustrates how financial damages caused by cyber incidents evolve over time, revealing increasing economic consequences of cyber threats.

---

### 🔍 Interactive Filters (Slicer Panel)

The dashboard includes slicers that allow users to dynamically filter data by:

* Year
* Country
* Attack Type
* Target Industry
* Attack Source

This enables deeper exploration of cybersecurity trends from multiple perspectives.

---

## Business Impact & Insights

The dashboard provides several valuable insights:

* **Ransomware and phishing attacks dominate the threat landscape**
* **IT, Finance, and Healthcare industries are the most targeted**
* **Nation-state actors and hacker groups contribute to major incidents**
* Financial losses from cyberattacks show an **increasing trend over time**
* Many attacks exploit **social engineering and unpatched software vulnerabilities**

These insights can help organizations:

* Improve cybersecurity strategies
* Prioritize risk mitigation efforts
* Enhance incident response planning
* Increase awareness of emerging cyber threats
